Моя сборка для верстки сайтов
=======

Подключено:
```
Bootstrap 3.0.3
Jquery 2.0.3
Modernizr 2.7.1
```

Установка linux Mint/ Ubuntu

```
git clone git@github.com:nikolaevav/static-page.git
cd static-page
bundle
```

Запуск сервера:

```
middleman
```

Открытие в браузере:

```
http://localhost:4567
```


Генерация сайта:

```
bundle exec middleman build
```
